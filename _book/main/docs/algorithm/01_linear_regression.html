線形回帰モデル

線形規程関数モデル(線形回帰)

回帰問題の目標はNNN個の観測値{xn}(n=1,...,N)\{x_n\}(n=1,...,N){x​n​​}(n=1,...,N)と， それに対応する目標値yyyの集合からなるデータ集合が与えられた時， 目標値が未知であるxxxが与えられた時の目標値yyyを予測するモデル

y=w0+w1x1+...+wDxD \LARGE y = w_0+w_1x_1+...+w_Dx_D y=w​0​​+w​1​​x​1​​+...+w​D​​x​D​​

式１：線形回帰モデル

y:xy:xy:xにおける目標値

D:D:D:入力の次元数(変数の数)

x:x:x:入力変数

w:w:w:パラメータ

単回帰

適当なデータを回帰してみましょう。この目では単回帰(変数が１つの問題)を扱います。求めるべき変数が、直線の傾きになりますね。
変数が１つというのは上の式のDDDが1ということなので

y=w1x+w0 \LARGE y = w_1x + w_0 y=w​1​​x+w​0​​

式２：単回帰モデル

y:xy:xy:xにおける目標値

x:x:x:入力変数

w1:w_1:w​1​​:直線yの傾き

w0:w_0:w​0​​:直線yの切片

ということになります。

見ての通り、データは直線では表しにくいものですので、新しいデータxxxに対して確実にコレ！というyyyを求めることは難しいです(ある点にはかなり近いが違う点からは遠い　など)。ですので、ある程度の誤差は仕方ありません。できる限り誤差を減らし、図２のようなそれらしい直線を引くことが単回帰分析となります。

誤差

どのくらいデータに合っているか、誤差が小さい直線かを判断する目安として、
平均二乗誤差関数(Mean Squared Error)があります。

MSE(y,y^)=1N∑n=0N−1(yn−y^n) \LARGE MSE(y,\hat{y}) = \frac{1}{N} \sum_{n=0}^{N-1} (y_n - \hat{y}_n) MSE(y,​y​^​​)=​N​​1​​​n=0​∑​N−1​​(y​n​​−​y​^​​​n​​)

式３：平均二乗誤差関数

y:y:y:目標値

y^:\hat{y}:​y​^​​:予測値

N:N:N:変数の数(サンプル数)

この平均二乗誤差関数(MSE)は、実際のデータと回帰した結果の関数との差分(図３の赤い矢印)を二乗したものであり、式３で表されます。二乗しているのは、誤差の正負を考慮しないためです。Nで割って平均にするのは、大きく外れた値によって誤差が大きくなるのをある程度防ぐためです。

図３：平均二乗誤差関数：各点と回帰直線の差分

イメージとしてはこれらの矢印をバネに見立てるという考え方になります。各点と回帰直線の間に同じ強さのバネをつけて、バネが動かなくなる点が、距離の二乗和の平均が最小になる(バランスが取れる)ところになるというわけです。
仮に各点と直線の距離が同じだとしたらバネは全て同じ長さで安定するはず、、、ですよね？
各点かかる力が平均的になるイメージ

編集後記

使用したサンプルデータ的なものは以下のコードで生成しました

np.random.seed(seed=1)
X_n = 16
X = 5 + 25 * np.random.rand(X_n)
Y=[]
noise = np.random.normal(-0.3,0.3,16)
X = X + noise
for i in X:
  Y.append(math.log(i))

Y = Y + noise

X : 0~1の乱数に25を掛けて5を足したものを16個
Y : Xの対数に、-0.3 ~ 0.3 の間で生成したランダムなノイズを足したもの
